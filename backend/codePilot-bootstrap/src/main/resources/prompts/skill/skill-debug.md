name: debug
description: Debug an error or unexpected behavior systematically

## Steps

1. REPRODUCE the error by identifying the exact steps, inputs, or state.
2. ISOLATE the root cause by:
   - Reading the related source code
   - Checking logs / error messages
   - Using grep to find relevant code patterns
3. IMPLEMENT the fix.
4. VERIFY the fix by running the reproduction case.
5. CHECK that no related code is broken.
